# OZAFLIX - Frontend Project

## Description
A frontend web application project built with HTML, CSS, JavaScript, and Tailwind CSS.  
Includes interactive features such as watchlist, profile management, and movie details.

## Features
- Responsive UI design
- DOM manipulation & dynamic content
- Form validation
- Navigation between pages
- Organized folder structure

## Technologies
- HTML5
- CSS3 / Tailwind CSS
- JavaScript (vanilla)
